Reports Screen

The Reports screen allows you to view individual reports and to create customized version of the reports for your own use or to share with other users. The screen is divided into two main sections, the Reports Tree and the Report Viewer.

Reports List

The Reports List shows all of the reports that are available for viewing. The reports are divided into the following categories:

●  My Reports - The Key Performance Indicators (KPI) Report and any customized reports you have created are listed under this category.

●  Customer Profiling -

●  Error Management -

●  Network Performance -

●  Shared Reports - Any reports created by other users ans shared are listed in under this category.

For more information on a report, see Reports.

Report Viewer

The Report Viewer displays the currently selected report. This area of the screen is blank until a a report is selected from the Report List.

Edit Report Parameters

The Edit Report Parameters button displays Edit Report Parameters dialog. The Edit Report Parameters dialog allows you to change the parameters for the selected report.
