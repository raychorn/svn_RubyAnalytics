Navigation Area

The Navigation area contains links to the other functions of the Analytics Portal.

The navigation area is divided into three sections:

●  Primary Navigation Bar — The list of links at the top of the Navigation Area are global links. The links displayed to each user in this navigation bar depend on the user's Role. For more information on roles, see Roles Screen. Depending on your role, the following links may appear on the Primary Navigation Bar:

●  Analytics — Select this link to access the Dashboard, Reports, and Data Segments screens.

●  Users — Select this link to access the Users screen.

●  Roles — Select this link to access the Roles screen.

●  Config — Select this link to access the Configuration screen.

●  Secondary Navigation Bar — This list of links appears below the Primary Navigation bar. The links listed in this area depend on what is selected in the Primary Navigation Bar.

●  Tool Bar — The Tool Bar contains three links:

●  Username — Click on your name to access the My Account screen.

●  Log Out — Select this link to log out of the Analytics Portal.

●  Help — Select this link to access the online help system.
