Technology Performance Summary Report

The Technology Performance Summary Report provides you with a summary of information about the performance of the connection technologies used on your network during the selected time range. The Technology Performance Summary Report appears on the Reports screen under the Network Performance category.

Type

Tabular Data

Parameters

●  Time Range – default to past week

●  Default sort by Connections/day descending

Fields

●  Connection Type

●  Connections/day

●  Failure Rate (%) – (Failed Connects + Drops) / (Failed Connects + Drops + Successful Connects) * 100. DataParam.

●  Throughput/day – Calculate average data uploaded and downloaded per day.

●  Average Peak Download Speed – Calculate average peak download speed.

●  Average Signal Strength

Sample

Sorting

●  Conn. Type ASC

●  Connects/day DEC, Failure Rate DEC, Throughput/day DEC

●  Failure Rate DEC, Connects/Day DEC, Throughput/day DEC

●  Throughput/day DEC, Avg Peak Down Sp DEC, Avg Signal DEC

●  Avg Peak Down Sp DEC, Throughput/day DEC, Avg Signal DEC

●  Avg Signal DEC, Throughput/day DEC, Avg Peak Down Sp DEC
