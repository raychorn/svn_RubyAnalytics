Connections per Day Heat Map Report

The Connections per Day Heat Map shows you the number of connections per day during a specified time range as an overlay on a map. The darker the block, the greater the number of daily connections in that area. The Connections per Day Heat Map appears on the Reports screen under the Network Performance category.

Type

Map

Parameters

●  Time Range — The default time range is set to the past month.

●  Connection Type

Data

Display a heat map of the number of connections per day. Base station markers are included on the map.
