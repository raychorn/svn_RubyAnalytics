Daily Usage Trend

The Daily Usage Trend report shows the average usage for each day in order to uncover behavioral patterns during the day. For the selected Time Range, the chart shows the average values for each hour of the day. For example, the first value, Midnight to 1 am, shows the average of all the data collected between Midnight to 1 am throughout the entire Time Range. The Daily Usage Trend report appears on the Reports screen under the Customer Profiling category.

Type

Mixed – area and line graphs

Parameters

●  Time Range – The default range is the past month.

●  Connection Type

Data

X-axis

●  Hours of day, Midnight to Midnight, using end-user’s local time zone.

Y-axis

●  Line graph - Connections/hour

●  Line graph – Failure Rate (%) - (Failed Connects + Drops) / (Failed Connects + Drops + Successful Connects) * 100.

●  Area graph - Average data uploaded and downloaded
