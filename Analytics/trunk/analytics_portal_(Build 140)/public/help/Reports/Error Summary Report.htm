Error Summary Report

The Error Summary Report provides you with a summary of information about the errors that were reported by your network during the selected time range. The Error Summary appears on the Reports screen under the Error Management category.

Type

Tabular

Parameters

●  Time Range – default to past week

●  Default sort by % Users Affected descending.

Fields

●  Description (Error Code) – Format description and error code into a single field.

●  % Users Affected – Calculate the percent of total active users that experienced this error at least once.

●  Frequency/hour – Divide the total count of error instances by the total connected time for affected users (not all users) during the report time range.

Sample

Sorting

●  Error ASC

●  % Users DEC, Freq/hr DEC, Error ASC

●  Freq/hr DEC, Error ASC, % Users DEC
