Error Scatterplot Report

The Error Scatterplot Report shows you graphically the severity of errors that are occurring on your network. The size of the points that appear on the chart and where they are located help you to determine the relative severity of the errors. The higher up and further right a point appears the more frequently the error occurs and more users are affected. The size of the point indicates the total number of times an error has occurred. The Error Scatterplot Report appears on the Reports screen under the Error Management category.

Type

Scatterplot

Parameters

●  Time Range – default to past month

Data

X-axis

●  Frequency/hour – Divide the total count of error instances by the total connected time for affected users (not all users) during the report time range.

Y-axis

●  % Users Affected – Calculate the percent of total active users that experienced this error at least once.

Point Sizes (relative radius of each plotted point)

●  Total Count of each error

Point Tooltip

When the user hovers over a specific data point, a callout appears showing:

●  Error Code

●  Description

●  Connection Type
