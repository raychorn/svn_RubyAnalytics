Getting Started

The Smith Micro Analytics Portal provides you with powerful tool for measuring and analyzing customers' behaviors. This application allows you to view the way your customers use your network graphically and in tabular forms. Quick views of target audience data are provided via Dashboards. Detailed views are available via Reports. Your data can be segmented to provide granular views of particular target audiences via Data Segments.

Data Segmentation

Data segmentation is a key part of the Analytics Portal. It is one of the most important things that needs to be done to get the most out of the Analytics Portal.You need to segment your data in such a way that you can view focused information. Initially, all of your data is grouped together in one unfiltered data segment, the All Customers data segment. In order to narrow the focus of your analysis, you need to further segment this data so that any data you are not interested in is filtered out. Your data can be filtered by any combination of the following filters:

●  Network Technology

●  Device

●  Location

●  Vendor

●  Connection Frequency

●  Throughput

By using these filters, you can create data segments that focus on target audiences or customer experiences that you are interested in analyzing. For more information on data segmentation, see Data Segments Screen.

Dashboards

Each user of the Analytics Portal can create up to five customized dashboards that are associated with different data segments. When you create a dashboard you can customize parameters for each of the report widgets that appear on the dashboard. For more information on dashboards, see Dashboard Screen.
