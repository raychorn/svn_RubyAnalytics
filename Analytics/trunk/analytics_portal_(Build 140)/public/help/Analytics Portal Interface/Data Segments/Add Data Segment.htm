Add/Edit Data Segment

The Add/Edit Data Segment dialog appears when you click the Add New Data Segment button or the Edit button associated with a data segment you created. This dialog allows you to set the filters that will focus your new data segment on the information you want to analyze. The Add/Edit Data Segment dialog has four parts:

●  General

●  Network/Device Filter

●  Location/Vendor Filter

●  Usage Filter

General

Enter/edit the segment name and description in this part. The name and description you enter here are displayed in the Data Segments screen.

Network/Device Filter

Select/edit the network technology and devices that are part of the data segment in this part.

Location/Vendor Filter

Select/edit the locations and vendors that are part of the data segment in this part.

Usage Filter

Select/edit the connection frequencies and throughputs that are part of the data segment in this part.
