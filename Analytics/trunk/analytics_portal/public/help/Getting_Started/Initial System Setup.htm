Initial System Setup

The first time you run the Smith Micro Analytics Portal, you will need to do initial system setup. The following dialog is displayed at startup:

Do the following to complete initial system setup:

1. Enter the Customer Name.
2. Enter the first name of person that will be the Global Administrator in the Global Admin First Name field.
3. Enter the last name of person that will be the Global Administrator in the Global Admin Last Name field.
4. Enter the email address of person that will be the Global Administrator in the Global Admin Email field. The email address you enter here will server as your user ID.
5. Enter a password in the Password

●  Minimum length of at least 8 characters.

●  Must contain at least 1 letter, 1 number, and 1 symbol.

6. Re-enter the password in the Confirm Password field.

Note: The values entered in the Pasword and Confirm Password fields must match.

7. Click Setup.

Initial system setup is complete. The Analytics Portal Dashboard is displayed.
