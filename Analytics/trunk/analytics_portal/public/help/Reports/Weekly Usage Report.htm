Weekly Usage Trend Report

The Weekly Usage Trend Report shows the average usage for each day of the week in order to uncover behavioral patterns during each week. For the selected Time Range, the chart should calculate average values for each day of the week. For example, the Monday value shows the average of all the data collected on all Mondays throughout the entire Time Range. The Weekly Usage Trend Report appears on the Reports Screen under the Customer Profiling category.

Type

Mixed – area and line graphs

Parameters

●  Time Range – default to past month

●  Connection Type

Data

X-axis

●  Days of week, Monday through Sunday, using end-user’s local timezone.

Y-axis

●  Line graph - Connections/day

●  Line graph – Failure Rate (%) - (Failed Connects + Drops) / (Failed Connects + Drops + Successful Connects) * 100.

●  Area graph - Average data uploaded and downloaded)
