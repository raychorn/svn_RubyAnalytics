Data by Technology Report

The Data by Technology report is a stacked area graph that shows the average data throughput (upload and download) by technology (3G, 4G, Wi-Fi, Ethernet), The default time range is the previous year, but this parameter as well as which technologies are shown can be modified. The Data by Technology report appears on the Reports Screen under the Customer Profiling category.

Type

Stacked Area Graph

Parameters

●  Time Range – default to past year

Data

X-axis

●  Time, show past year.

Y-axis

●  Average data uploaded and downloaded for each connection type
