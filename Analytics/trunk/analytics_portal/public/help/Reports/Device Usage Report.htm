Device Usage Report

The Device Usage Report displays key information about each of your devices in a tabular format. The Device Usage Report appears on the Reports Screen under the Customer Profiling category.

Type

Tabular

Parameters

●  Time Range – default to past month

●  Error Code

●  Connection Type

●  Default sort by Share descending.

Columns

●  Manufacturer

●  Device Model

●  Share – Calculate the percent of total users with this device/OS.

●  Users – Calculate number of users with this device/OS.

●  Time Connected/day – Calculate average time connected per day of users with this device/OS.

●  Data Upload/day – Calculate average data uploaded per day of users with this device/OS.

●  Data Download/day – Calculate average data downloaded per day of users with this device/OS.

●  Average Peak Download Speed – Calculate average peak download speed of users with this device/OS.

●  Average Peak Upload Speed – Calculate average peak upload speed of users with this device/OS.

Sample

Sorting

●  Manufacturer ASC, Device ASC

●  Device ASC

●  Share DEC, Time Conn/day DEC, Data Dwn/day DEC

●  Users DEC, Time Conn/day DEC, Data Dwn/day DEC

●  Time Conn/day DEC, Share DEC, Data Dwn/day DEC

●  Data Dwn/day DEC, Data Up/day DEC, Share DEC

●  Data Up/day DEC, Data Dwn/day DEC, Share DEC

●  Avg Peak Dwn Sp DEC, Avg Peak Up Sp DEC, Share DEC

●   Avg Peak Up Sp DEC, Avg Peak Dwn Sp DEC, Share DEC
