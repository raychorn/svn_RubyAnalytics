Error Report

The Error Report provides you with key pieces of information about all the errors that were reported by your network during the selected time range. The Error Report appears on the Reports screen under the Error Management category.

Type

Tabular

Parameters

●  Time Range – default to past week

●  Default sort by % Users Affected descending

Columns

●  Error Code

●  Description

●  Connection Type

●  % Users Affected – Calculate the percent of total active users that experienced this error at least once.

●  3-month Trend of % Users Affected – Chart values for each day. Mark high and low points.

●  Frequency/hour – Divide the total count of error instances by the total connected time for affected users (not all users) during the report time range.

●  Total Count of each error

Sample

Sorting

●  Error Code ASC

●  Description ASC, Error Code ASC

●  Conn Type ASC, % Users DEC, Freq/hr DEC

●  % Users DEC, Freq/hr DEC, Total DEC

●  Freq/hr DEC, % Users DEC, Total DEC

●  Total DEC, % Users DEC, Freq/hr DEC
