Usage Trend Report

The Usage Trend Report shows you three key pieces of information: the number of connections per day for all technologies, the percentage of failures per day, and the average data uploaded and downloaded per day. The Usage Trend Report appears on the Reports Screen under the Customer Profiling category.

Type

Mixed – area and line graphs

Parameters

●  Time Range – default to past year

●  Connection Type

●  Manufacturer

●  Device Model

Data

X-axis

●  Time, shows the past year by default.

Y-axis

●  Line graph - Connections/day

●  Line graph – Failure Rate (%) - (Failed Connects + Drops) / (Failed Connects + Drops + Successful Connects) * 100. Area graph - Average data uploaded and downloaded
