Dashboard Screen

The Dashboard screen is the primary screen displayed for all users. The default dashboard that appears for all users is associated with the unfiltered All Customers data segment. Each user can create up to five additional dashboards that are associated with other Data Segments Screen.

Each dashboard screen is comprised of report widgets that display some of the key reports available from the Analytics Portal. The following report widgets appear on dashboards:

●  Weekly Usage — The Weekly Usage Trend Report shows the average usage for each day of the week in order to uncover behavioral patterns during each week. For the selected Time Range, the chart should calculate average values for each day of the week. For example, the Monday value shows the average of all the data collected on all Mondays throughout the entire Time Range.

●  Device Usage Pie Chart— The Device Usage Pie Chart allows you to see in relation to each other, which devices your users are using.

●  Key Performance Indicator — The KPI Report allows you to select specific data elements to track. Each of these data elements is can have a target value. If the value of a data elements raises above or falls below you target value, you can be notified of the event.

A user can edit the parameters displayed in each of the report widgets.
