Data Segments Screen

The Data Segments screen allows you to create, edit, delete, and update data segments. The main parts of content area of the Data Segments screen are highlighted in the image below.

Data segments are filtered views on the baseline data segment (All Customers). They allow you to create a slice of data that is used by reports and dashboards.

All Customers Data Segment

Initially, only the baseline data segment (All Customers) is listed on the Data Segments screen. The All Customers data segment contains all of your customer data. This data segment cannot be edited or deleted. The only function that can be performed on this segment is to update it.

User Created Data Segments

You can create data segments to focus your data on just the target audiences or customer experiences that you are interested in analyzing. Data Segments can be created in two ways:

●  A data segment is automatically created for you when you copy another user's shared dashboard. For more information on adding dashboards, see Add/Edit Dashboard.

●  You can create your own data segments by clicking on the Add New Data Segment button. For more information on the Add/Edit Data Segment dialog, see Add/Edit Data Segment.
